Colorful Traces

by Alexandre Dumur and Raphael Strebel

Data presentation

In this section we show a brief global description of the data.
The data was provided by the Smood affiliation and consists of a set of points which are shown according to their geographical locations. The points can be either pickup locations, dropoff locations or points on a road from a pickup to a dropoff. We also have access to the time a driver needed for any path.
There are almost 12'600 nodes shown in the networks below, 42 of which correspond to pickups and 525 are dropoffs. There are also 2000 roads in the data, although some of them overlap eachother.
Our goal is focus on a visualisation of this geographical data without using the distraction of a map in the background. We attempt to get as much as we can out of the information provided, and show that it can be nice to watch and interesting to interact with.

Black Lightning

The first network you see is a representation of all nodes of the database, colored according to their occurrences. The coloring is done using a linear scale from a dark blue (#112231) for 0 occurrences going through a lighter blue (#3C769D) when the number of occurrences reaches 40 and to white linearly.

Interactive Network

Below you can see the interactive network we built. You can click on the red pickup nodes to see all the paths that start at this location. You can also click on a dropoff node to display all paths to this point. The paths shown in green appear according to the real time data, where one second in the visualization corresponds to one minute for the driver. On the side you can also see some interesting information about the node you chose, for example it's location on a map.
