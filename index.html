Black Lightning

The first network you see is a representation of all nodes of the database, colored according to their occurrences. The coloring is done according to a linear scale from "#112231" to "#3C769D".

Interactive Network

Below you can see the interactive network we built. You can click on the red pickup nodes to see all the paths that start at this location. You can also click on a dropoff node to display all paths to this point. The paths shown in green appear according to the real time data, where one second in the visualization corresponds to one minute for the driver. On the side you can also see some interesting information about the node you chose, for example it's location on a map.
